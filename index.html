Miyabi Test の特徴

🤖

自律型エージェント

IssueAgent、CodeGenAgent、ReviewAgentなど、複数のAIエージェントが協調してタスクを自動実行します。

🔄

GitHub Actions統合

AI自動ラベリング、自動コード生成、自動レビューなど、包括的なCI/CDワークフローを提供します。

📊

週次レポート

プロジェクトのKPIやメトリクスを自動集計し、週次レポートとして配信します。

🔐

セキュリティスキャン

CodeQLによる自動セキュリティスキャンと、Dependabotによる依存関係の自動更新を実装。

🚀

自動デプロイ

GitHub PagesへのデプロイやDockerイメージのビルド・プッシュを自動化します。

📡

Webhookルーティング

GitHubイベントを受信し、適切なエージェントにルーティングするイベントバスを実装。
